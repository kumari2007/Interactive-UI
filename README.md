# Interactive Login & Registration UI

A modern and responsive Login & Registration User Interface developed using HTML, CSS, Bootstrap, and JavaScript.

## 🚀 Features

- Responsive Login Page
- Responsive Registration Page
- Modern Neon UI Design
- Strong Password Validation
- Interactive Popups using JavaScript
- User-Friendly Interface
- Bootstrap Icons Integration

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## 🔐 Password Validation

The registration form checks for:
- Minimum 8 characters
- Uppercase letter
- Lowercase letter
- Number
- Special character
